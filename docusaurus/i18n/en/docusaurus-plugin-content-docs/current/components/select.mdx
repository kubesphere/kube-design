# Select

A dropdown select component for selecting values from a list.

## When To Use

- Use when you need to select a single value from a list
- Supports search/filter functionality
- Can display custom content for options

## Examples

### Basic Usage

Basic select dropdown.

```jsx
import { Select } from '@kubed/components';

function Demo() {
  const options = [
    { label: 'React', value: 'react' },
    { label: 'Vue', value: 'vue' },
    { label: 'Angular', value: 'angular' },
  ];

  return (
    <Select
      placeholder="Select framework"
      options={options}
    />
  );
}
```

## API

### Select Props

| Property | Description | Type | Default |
|----------|-------------|------|---------|
| value | Selected value | `string \| string[]` | - |
| defaultValue | Default value | `string \| string[]` | - |
| placeholder | Placeholder text | `string` | - |
| options | Select options | `Option[]` | `[]` |
| disabled | Disabled state | `boolean` | `false` |
| searchable | Enable search | `boolean` | `false` |
| multiple | Multiple selection | `boolean` | `false` |
| onChange | Change handler | `(value: string \| string[]) => void` | - |

### Option

```typescript
interface Option {
  label: string;
  value: string;
  disabled?: boolean;
}
```
