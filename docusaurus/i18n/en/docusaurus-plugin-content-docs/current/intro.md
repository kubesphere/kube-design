## Quick Start

Get started with Kube Design in minutes:

```bash
# Install packages
npm install @kubed/components @kubed/hooks @kubed/icons
# or
yarn add @kubed/components @kubed/hooks @kubed/icons
# or
pnpm add @kubed/components @kubed/hooks @kubed/icons
```
