# Icon Usage

Learn how to use icons in your Kube Design applications.

## Installation

Icons are included in the `@kubed/icons` package:

```bash
npm install @kubed/icons
# or
yarn add @kubed/icons
# or
pnpm add @kubed/icons
```

## Basic Usage

Import icons individually for better tree-shaking:

```jsx live
function Demo() {
  return (
    <Group spacing="md">
      <Add size={24} />
      <Close size={24} />
      <Check size={24} />
      <Delete size={24} />
    </Group>
  );
}
```

## Icon Sizes

Control icon size using the `size` prop:

```jsx live
function Demo() {
  return (
    <Group spacing="lg">
      <Add size={16} />
      <Add size={24} />
      <Add size={32} />
      <Add size={48} />
    </Group>
  );
}
```

## Icon Colors

Icons inherit the current text color:

```jsx live
function Demo() {
  return (
    <Group spacing="md">
      <Add size={32} style={{ color: '#55bc8a' }} />
      <Add size={32} style={{ color: '#329dce' }} />
      <Add size={32} style={{ color: '#f5a623' }} />
      <Add size={32} style={{ color: '#ca2621' }} />
    </Group>
  );
}
```

## In Buttons

Icons work seamlessly with Button components:

```jsx live
function Demo() {
  return (
    <Group spacing="md">
      <Button leftIcon={<Add />} color="primary">
        Add Item
      </Button>
      <Button rightIcon={<Delete />} color="error">
        Delete
      </Button>
      <Button variant="outline">
        <Check />
      </Button>
    </Group>
  );
}
```

## Available Icons

Kube Design includes hundreds of icons. Check the [overview](/docs/icons/overview) for a complete list.

## API

### Icon Props

| Property | Description | Type | Default |
|----------|-------------|------|---------|
| size | Icon size in pixels | `number \| string` | `16` |
| color | Icon color (CSS color value) | `string` | `'currentColor'` |
| className | Additional CSS class | `string` | - |
| style | Inline styles | `CSSProperties` | - |
